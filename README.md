## About the Project

**Landify Landing Page** is a front-end web design project developed to practice real-world landing page structure, responsive layouts, custom styling, Bootstrap grid usage, and professional UI section building.

The goal of this project is to create a visually attractive and easy-to-use landing page that looks good on desktop, tablet, and mobile screens.

## Features

- Modern and clean landing page design
- Fully responsive layout
- Bootstrap-based responsive navbar
- Hero section with mobile mockup
- Brand/logo marquee animation
- Tailor-made feature cards section
- Customer testimonial cards
- Achievement/statistics section
- Product highlight section
- Large image showcase section
- Mobile app download call-to-action
- Professional footer with useful links
- Hover effects on buttons and links
- Organized assets folder structure

## Tech Stack

## Sections Included

### 1. Navbar
A responsive Bootstrap navbar with brand logo, menu links, dropdown menu, and app download badges.

### 2. Hero Section
A strong landing page introduction with headline, call-to-action button, background design, and phone mockup image.

### 3. Brand Marquee
Animated brand/logo strip showing companies such as Airbnb, HubSpot, Google, Microsoft, Walmart, and FedEx.

### 4. Features Section
A clean feature grid showing important product qualities like workflow, flexibility, user-friendly design, multiple layouts, better components, and organization.

### 5. Testimonials
Customer story cards with brand logos, user feedback, names, and roles.

### 6. Achievements
A statistics section showing downloads, users, clients, and countries.

### 7. Product Highlight
A two-column section with mobile mockup and supporting product description.

### 8. CTA Section
A call-to-action section encouraging users to download the app from Google Play or the App Store.

### 9. Footer
A dark professional footer with navigation links, copyright text, and app download buttons.

## How to Run the Project

This is a static front-end project, so no installation is required.

### Option 1: Open Directly

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` to open it in your browser.

### Option 2: Run with VS Code Live Server

1. Open the project in VS Code.
2. Install the **Live Server** extension.
3. Right-click on `index.html`.
4. Click **Open with Live Server**.
